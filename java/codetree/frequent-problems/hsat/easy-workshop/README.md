# [편안한 워크숍](https://www.codetree.ai/ko/frequent-problems/hsat/problems/easy-workshop/description)

H사의 신입사원 A는 입사동기들과 워크숍에서 등산을 가기로 하였습니다. 등산하려는 산의 정보는 N×N 격자로 주어지며 각 칸에 적혀있는 수들은 높이를 의미합니다. A는 입사동기들의 의견을 존중하며 그 중 가장 편안한 등산로를 찾기 원합니다. 입사동기들의 의견을 만족하는 등산로 중 인접한 높이의 차들 간의 최댓값이 최소가 되는 등산로가 가장 편안한 등산로입니다.

다음의 조건들은 입사동기들의 의견입니다.

1. 등산로는 어느 칸에서 시작해도 상관 없습니다.
2. 등산로는 시작칸으로부터 상하좌우로 인접한 칸으로만 이동하는 것을 원칙으로 하며, 이동할 때마다 높이가 높아져야 합니다.
3. 등산로의 길이는 K 이상이어야 합니다. 등산로의 길이는 해당 등산로로 갈 때 지나가는 칸의 수를 의미합니다.

A를 도와 입사동기들의 의견을 만족하며 가장 편안한 등산로를 찾는 프로그램을 작성하세요.
